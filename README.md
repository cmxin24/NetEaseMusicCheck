# NetEaseMusicCheck (网易云音乐自动签到助手)

## 简介

基于 Chrome Manifest V3 的浏览器扩展，旨在帮助用户自动完成网易云音乐的每日签到任务，轻松获取积分。整个过程在后台静默运行，无需人工干预，不打扰日常浏览体验。

## 主要功能

- **自动签到**：每天自动尝试打开网易云音乐网页端签到。
- **定时执行**：每日定时签到（通常在每日凌晨 00:00:05 左右），如果浏览器未打开或处于休眠状态，将在下一次打开时执行。
- **后台运行**：无需手动点击，签到过程在后台静默完成，不会弹出通知。

## 安装方法

1. **下载**：点击右侧Releases，下载最新版本crx文件。
2. **打开扩展管理**：点击菜单-->扩展程序-->管理扩展程序。或在浏览器地址栏输入 `chrome://extensions/` (Chrome)  `edge://extensions/` (Edge) `brave://extensions/`(Brave)并回车。
3. **安装扩展**：将crx插件拖入页面。

## 使用说明

1. **登录账号**：请确保您已经在 Chrome 浏览器中访问过 [网易云音乐官网 (music.163.com)](https://music.163.com/) 并成功**登录**您的账号。
2. **自动运行**：扩展程序会自动读取您的登录状态（Cookies），并在后台自动进行每日的签到任务。
4. **状态切换**：不需要时点击logo即可进入停用状态，logo将自动切换为灰色。
3. **海外用户**建议搭配[NetEaseMusicWorld++](https://github.com/kogamitora/NetEaseMusicWorldNext)使用，打开网页后该账号当天会被视为中国ip，手机端也会同步解锁歌曲权限。
